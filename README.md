# 🚗 Car Price Prediction

A Machine Learning web application that predicts car prices based on vehicle features. The project uses data preprocessing, model selection, and a Gradio interface for user interaction.

## 📌 Features

- Data preprocessing pipeline
- Multiple ML model comparison
- Best model selection
- Car price prediction
- Interactive Gradio web app

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio
- Joblib/Pickle

## 📂 Project Structure

```
car_price_project/
│
├── app.py
├── model.py
├── model.pkl
├── CarPrice_Assignment.csv
├── requirement.txt
└── README.md
```

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/ibmrafi69/car-price-prediction.git
cd car-price-prediction
```

Install dependencies:

```bash
pip install -r requirement.txt
```

Run the application:

```bash
python app.py
```

## 📊 Dataset

The project uses the CarPrice Assignment dataset containing vehicle specifications and corresponding prices.

## 🤖 Machine Learning Workflow

1. Data Cleaning

4. Model Training
5. Model Evaluation
